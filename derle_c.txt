Güncelleme Tarihi: 29.08.2024

Bu belge, Elera İşletim Sistemi çekirdeğinin derlenmesi için gerekli yönergeleri içerir.

Derleme Ortamı:
---------------
  Sistem: Windows 7 - (64 / 32 bit)
  Derleyici: Lazarus 2.0.8, FPC 3.0.4
  Sanal Makine: Virtualbox 7.0.20

Ön kurulum:
---------------
32 bit windows:

  + lazarus-2.0.8-fpc-3.0.4-win32.exe programını c:\lazarus klasörüne kurunuz
  
64 bit windows ortamında:

  + lazarus-2.0.8-fpc-3.0.4-win64 programını c:\lazarus klasörüne kurunuz
  + lazarus-2.0.8-fpc-3.0.4-cross-i386-win32-win64 programını c:\lazarus klasörüne kurunuz
  

RTL Derleme - 32 bit windows
-----------
  + elerais\araclar\binutils dizininin altındaki sıkıştırılmış binutils.7z dosya içeriğindeki *.exe dosyalarını c:\lazarus\fpc\3.0.4\bin\i386-win32 dizinine kopyalayınız.
  + elerais\rtl_cekirdek dizininin altındaki derle_x86.bat dosyasını çalıştırarak RTL'yi derleyiniz.

RTL Derleme - 64 bit windows
-----------
  + elerais\araclar\binutils dizininin altındaki sıkıştırılmış binutils.7z dosya içeriğindeki *.exe dosyalarını C:\lazarus\fpc\3.0.4\bin\x86_64-win64 dizinine kopyalayınız.
  + elerais\rtl_kernel dizininin altındaki derle_x64.bat dosyasını çalıştırarak RTL'yi derleyiniz.

Çekirdek (Kernel) Derleme - 32 bit windows
-------------------------
  + elerais\cekirdek\cekirdek.lpr dosyasını isterseniz lazarus'ta Ctrl+F9 ile, isterseniz ilgili klasörün altındaki derle_x86.bat dosyasını çalıştırarak derleyiniz.

Çekirdek (Kernel) Derleme - 64 bit windows
-------------------------
  + elerais\cekirdek\cekirdek.lpr dosyasını isterseniz lazarus'ta Ctrl+F9 ile, isterseniz ilgili klasörün altındaki derle_x64.bat dosyasını çalıştırarak derleyiniz.

Diğer Bilgiler
--------------
  + adresler.bat dosyasını, derleyicinin cekirdek.bin dosyasına atadığı adres değerlerini görmek için,
  + asm_kod.bat dosyasını, cekirdek.bin dosya içeriğinin assembler kodlarını görmek için,
  + derle_x86.bat/derle_x64.bat dosyasını, cekirdek.lpr çekirdek kodlarını derlemek için,
  + kopyala.bat dosyasını, cekirdek.bin dosyasını floppy (a: sürücüsü) aygıtına kopyalamak için,
  + temizle.bat dosyasını gereksiz dosyaları silmek için kullanabilirsiniz.
